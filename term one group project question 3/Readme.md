Question 3 — Word Counter

The Word Counter program allows the user to enter a sentence and then analyzes it.

 The program performs the following tasks:

- Counts the number of words
- Counts the number of characters (excluding spaces)
- Finds the longest word in the sentence
- Validates input to ensure it is not empty

 How the Code Works

1. The user is prompted to enter a sentence.
