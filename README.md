Project Description
This project is designed for **Introduction to Python Programming** and demonstrates key skills in:
- Python basics and problem solving  
- Functional programming with well-organized functions  
- Input validation and user-friendly interaction  
- Collaborative coding using Git and GitHub  
- Building a menu-driven console application  
The project consists of **four core programs** integrated into a **menu system**:
1. Number Analyzer – Checks if a number is even/odd, positive/negative, and prime.  
2. Grade System – Accepts 5 marks, calculates the average, and outputs the corresponding grade.  
3. Word Counter – Accepts a sentence and calculates:
   - Total number of words  
   - Number of characters (excluding spaces)  
   - Longest word in the sentence  
4. List Manager – Allows input of multiple numbers and outputs:
   - Largest number  
   - Smallest number  
   - Sorted list  
The menu system allows the user to select any program and return to the menu until choosing to exit.
